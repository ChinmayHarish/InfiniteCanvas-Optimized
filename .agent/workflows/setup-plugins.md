# Setup Plugins

This workflow verifies the installation of the essential plugins.

1.  **GSD Status**
    - Checks if `.gsd` directory exists.
    - Checks if `.agent/skills/planner` exists.

2.  **Ralph Loop Status**
    - Checks if `.agent/skills/ralph-loop` exists.
    - Checks if `.agent/workflows/ralph-loop.md` exists.

3.  **CodeRabbit Status**
    - Checks if `.agent/workflows/coderabbit.md` exists.

4.  **Confirmation**
    - Output the status of all plugins.
